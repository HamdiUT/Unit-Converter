#  Convert - Unit Converter

Un convertisseur d'unités moderne, élégant et intuitif construite avec HTML, CSS et JavaScript pur.

##  Fonctionnalités

- ** Conversion de température** : Convertissez instantanément entre Celsius et Fahrenheit
- ** Conversion de poids** : Convertissez entre Kilogrammes et Livres
- ** Conversion de distance** : Convertissez entre Kilomètres et Miles
- ** Design Responsive** : Fonctionne parfaitement sur tous les appareils
- ** Interface Moderne** : Design contemporain avec animations fluides
- ** Gradient Élégant** : Interface visuelle attrayante avec dégradés et ombres


### Installation
1. Clonez ou téléchargez ce projet
2. Ouvrez le fichier `index.html` dans votre navigateur
3. C'est tout ! 


### Technologies
- **HTML5** : Structure sémantique
- **CSS3** : Flexbox, Grid, Gradients, Animations
- **JavaScript ES6** : Conversion en temps réel
- **Google Fonts** : Typographie Inter

## Utilisation

1. Sélectionnez le type de conversion via la navigation
2. Entrez une valeur dans le premier champ
3. Cliquez sur le bouton "Convert"
4. Résultat apparaît instantanément dans le second champ
